# CLAUDE.md Doctor

**Live tool: https://ddolboghi.github.io/claudemd-doctor/**

Lint your `CLAUDE.md` / `AGENTS.md` (score 0–100, specific fixes), or generate a project-tailored one from a 2-minute form. 100% client-side — nothing you paste ever leaves your browser tab. Works offline.

## Why

Agent instruction files rot: they fill up with generic advice the model already knows ("write clean code"), vague adverbs ("handle properly"), IMPORTANT-spam, stale file paths, and sometimes real API keys. Every wasted line is context the agent doesn't spend on your actual task. The Doctor flags all of it with line numbers and concrete rewrites — 15 rules across bloat, vagueness, missing essentials, and hygiene.

## Use it

1. Open the [live tool](https://ddolboghi.github.io/claudemd-doctor/)
2. Paste your CLAUDE.md → **Diagnose** (optionally paste your file tree to catch dead path references)
3. Or use the **Generate** tab to build one that scores A from scratch

No signup, no tracking, no server.
